# AI Resume Screener

An AI-powered Resume Screening project that analyzes resumes against a job description, extracts skills, compares requirements, and generates candidate scores.

## Features

- Resume skill extraction using LLM
- Job description matching
- Missing skills detection
- Candidate scoring
- Strong / Average / Weak classification
- Easy to customize

## Tech Stack

- Python
- LangChain
- Groq API
- Prompt Engineering
- Environment Variables (.env)
